# 鸿蒙应用开发社区踩坑资料汇编（近一年向）

## 使用建议

1. 先按**现象**（编译错误码、白屏、不刷新、装不上包）在对应分册中检索关键词。
2. 同一问题在不同 API 版本上结论可能不同，请核对项目的 `compileSdkVersion` / `compatibleSdkVersion`。
3. 社区方案若涉及关闭 SSL、改注册表等操作，请在理解风险后再用于**本地调试**，生产环境遵循安全规范。
